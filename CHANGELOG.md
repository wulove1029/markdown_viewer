# Changelog

## [1.15.0] - 2026-07-09

### Added
- PDF 螢光標記現在可以直接在畫布上右鍵刪除；螢光面板新增「刪除」按鈕；螢光筆模式支援 `Ctrl+Z` 撤銷最後一筆。

### Improved
- GUI 框架由 PyQt6 遷移至 PySide6（Qt 官方維護、LGPL 授權），功能與外觀不變。
- 內部架構重整：匯出、視窗狀態、更新流程拆分為獨立模組，並新增整合測試，行為不變。
- 刪除全部螢光標記後，不再殘留空的標記檔（.highlights.json）。

## [1.14.0] - 2026-07-09

### Added
- 新增偏好設定視窗（設定 → 偏好設定…）：外觀（主題、內容縮放）、匯出（PDF 頁面尺寸/方向）、行為（更新檢查、自訂 CSS）集中管理，沿用既有設定值。

### Improved
- 大檔效能：Markdown 渲染移至背景執行緒，開啟大型文件不再凍結介面；編輯模式即時預覽改為停止輸入後才重新渲染。
- 匯出品質：Word/PPT 表格內的粗體、斜體、行內程式碼格式現在會保留；圖片依實際尺寸輸出（小圖不再被放大）；遠端圖片下載加入逾時與總量保護，失敗時以占位文字呈現。

### Fixed
- 修正快速切換分頁時可能顯示錯誤文件內容的競態問題。
- 修正非同步渲染後捲動位置（切換分頁、存檔重載）無法還原的問題。

### Added
- 新增匯出 Word（.docx）功能：由 Markdown 直接產生原生可編輯的 Word 文件，支援標題、清單、粗斜體、表格、程式碼區塊與圖片；Mermaid 圖與數學式自動轉為圖片內嵌，中文正文使用微軟正黑體。
- 分頁可右鍵「移至新視窗」，把單一文件拆出成獨立閱讀器視窗。

### Improved
- 單一實例機制：再次開啟 .md 檔（雙擊或命令列）不再開出新視窗，改為在既有視窗新增分頁並帶到前景。
- 獨立（拆出）視窗關閉時不再覆蓋主視窗的分頁還原記錄。

## [1.12.0] - 2026-07-01

### Added
- 加入 Mermaid 視覺化流程圖建立器，可用按鈕新增起點、程序、決策、終點節點，並拖曳節點調整位置。
- 支援流程圖自動連線、手動連線、自動佈局、右側屬性面板，以及 Mermaid 原始碼與視覺畫布雙向同步。
- 新增甘特圖視覺化編輯器，可編輯區段、任務、日期、狀態與持續時間。
- 新增序列圖、類別圖、狀態圖、ER 圖的結構化視覺編輯器，支援表格化編輯並回寫 Mermaid 語法。
- Mermaid 工作區支援從 Markdown fenced block 選取圖表編輯，儲存時回寫原文件。

### Improved
- Mermaid 工作區加入深色/淺色主題切換，並改善深色模式下的可讀性。
- 流程圖畫布支援滑鼠滾輪縮放、空白處拖曳平移，以及 `Ctrl+0` 重設縮放。
- 強化 Mermaid 圖表模板、語法片段、即時預覽、錯誤提示，以及 SVG/PNG 複製與匯出流程。
